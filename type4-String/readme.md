# String

## 字符串常用方法及注意事项
* trim() 去掉首尾空格
* charAt() 取单个字符
* charCodeAt() 取字符的 Unicode 值
* fromCharCode() Unicode 编码转换成字符串

## 字符串常用场景
* 反转：通过 2 个指针，前后交换
* 判断回文串：使用 DP
