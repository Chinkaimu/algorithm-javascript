# 二分查找

## 主要思想
* 将数据分为 2 堆，看结果是在左堆还是右堆。在左堆，左堆继续二分，知道找到数据。
* 关键点：
  * 中间点辨别
  * 如何二分
  * 判断目标是在左边还是在右边

## 场景
* 在数组（一维/二维）中找目标
* 在数组（一维/二维）中找到目标插入
* 在树中找到目标

## 解题方法
* 初始化起点 start，终点位置 end。求基点 pivot。
* 判断什么情况在左边部分，什么情况在右边部分。在左边则重置 end = mid - 1； 在右边则重置 start = mid + 1。
* 循环（递归）处理左边/右边数据。
* start >= end 跳出循环（终止递归）。
